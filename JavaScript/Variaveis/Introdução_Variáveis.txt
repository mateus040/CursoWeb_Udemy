Variáveis JavaScript


Tipos de variáveis:
- string
- number
- boolean


Regras para declaração de variáveis
- Não podem ser iniciados com números, apenas com letras ou "_"
- Não podem ser utilizados caracteres especiais como "ç", "^", "~"
- Não podem ser iguais as palavras reservadas da linguagem


Declarando variáveis com a instrução "var"


Case sensitive
- var curso, var Curso e var CURSO são variáveis diferentes


Exemplos
- var nome, var AulaVariaveis, var _idade  (Maneira correta)
- var 123nome, var número  (Maneira incorreta)
